# 列印模板庫 | 免費可列印 HTML 模板 (Free Printable Templates)

> **48 份免費、高品質、可直接列印或存成 PDF 的 HTML 模板**，涵蓋日週月計畫、習慣追蹤、健康紀錄、工作日誌、會議記錄、學習筆記等常用場景。無需安裝任何軟體，用瀏覽器開啟即可列印。

**[🌐 線上預覽 / Live Demo](https://print-templates.tranquility-base.work)** | **[🐙 GitHub Repository](https://github.com/bruce-yang-422/print-templates)**

---

## 專案概覽

設計理念：「**螢幕預覽直覺，列印輸出極簡**」。所有模板皆針對實際紙本體驗最佳化，黑白灰階省墨、精確留白適合打孔裝訂。

專案架構分兩層：

- **網站介面**：首頁提供分類篩選與預覽卡片，視覺採 Apple 風格簡潔主題
- **列印模板**：實際輸出的紙本版面，優先考量省墨、裁切、打孔與裝訂需求

目前共提供 **48 份** HTML 模板，支援即時 SVG 預覽。

## 列印步驟 (How to Print)

1. 使用 **Chrome** 或 **Edge** 開啟模板頁面
2. 在側邊欄選擇「內容模式」與紙張尺寸（A4 / B5 / A5 / A4裁A5）
3. 點擊「列印 / 存 PDF」按鈕
4. ⚠️ **重要**：在瀏覽器列印對話框將「邊界 (Margins)」設為「**無 (None)**」

## 技術說明 (Engineering)

無框架、原生輕量：

- **`index.html` + `css/style.css`**：首頁篩選、預覽卡片、全站 UI
- **`js/template-size.js`**：尺寸切換、列印前處理、`@page size` 動態注入
- **`js/calendar-data.js` + `assets/2026_calendar.json`**：跨模板共用日曆邏輯（農曆、週次、台灣假日）
- **`previews/`**：SVG 向量預覽圖，300×214 卡片規格
