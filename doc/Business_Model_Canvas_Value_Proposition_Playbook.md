# **商業模式九宮格與價值主張分析指南**

這份指南結合了《獲利世代》（Business Model Generation）與《價值主張年代》（Value Proposition Design）的核心理論，協助創業家與企業管理者透過視覺化的架構，釐清企業如何創造、傳遞及獲取價值。

## **一、 什麼是商業模式？**

商業模式是一個完整的高效率運行系統，旨在透過整合內外部要素，滿足客戶需求並實現客戶價值最大化，同時達成持續營利目標。

### **商業模式四大面向**

1. **核心價值**：如何「為顧客創造價值」？  
2. **需求面**：如何「讓顧客知道你」？  
3. **供給面**：此價值的「供給來源」為何？  
4. **財務面**：此價值如何「獲利」？

## **二、 商業模式九宮格 (Business Model Canvas)**

九宮格由九個相互關聯的元素組成，建議發想順序如下：

| 順序 | 元素名稱 | 英文 / 縮寫 | 定義 |
| :---- | :---- | :---- | :---- |
| 1 | **目標客層** | Customer Segments (CS) | 劃分顧客，鎖定最主要的目標群體。 |
| 2 | **價值主張** | Value Proposition (VP) | 如何解決顧客問題、滿足顧客需要？ |
| 3 | **通路** | Channels (CH) | 顧客在哪裡知道我們、在哪裡購買？ |
| 4 | **顧客關係** | Customer Relationships (CR) | 建立何種關係、如何維繫？ |
| 5 | **收益流** | Revenue Streams (R$) | 提供價值能取得多少收益、如何付費？ |
| 6 | **關鍵資源** | Key Resources (KR) | 提供價值需要什麼樣的資產？ |
| 7 | **關鍵活動** | Key Activities (KA) | 公司營運前必辦的最重要事項。 |
| 8 | **關鍵合作夥伴** | Key Partnership (KP) | 獲取資源與完成活動需要的外部夥伴。 |
| 9 | **成本結構** | Cost Structure (C$) | 運作上述所有元素需要的成本。 |

## **三、 核心深入：目標客層與價值主張**

根據《價值主張年代》，這兩格是商模的核心，可透過「顧客素描」與「價值主張圖」進行對接。

### **1\. 顧客素描 (Customer Profile)**

從顧客的角度出發，描繪以下三點：

* **顧客任務 (Customer Jobs)**：顧客在工作中或生活中想完成的事（功能性、社交性、情緒性、輔助性）。  
* **顧客痛點 (Customer Pains)**：完成任務前、中、後遇到的困擾、障礙或風險。  
* **顧客獲益 (Customer Gains)**：顧客希望獲得的結果或利益（必要、預期、渴求、意料外）。

### **2\. 價值主張 (Value Proposition)**

對應顧客的需求，提出相應的方案：

* **產品與服務 (Product & Service)**：提供給顧客的所有實體或無形項目。  
* **解決痛點 (Pain Relievers)**：產品如何減少或消除顧客的困擾。  
* **創造效益 (Gain Creators)**：產品如何放大顧客的獲益或創造驚喜。

## **四、 其他關鍵元素細節**

### **通路 (Channels) 的五個階段**

1. **認知**：提高對產品的知名度。  
2. **評估**：協助顧客評估價值主張。  
3. **購買**：使顧客能順利購買。  
4. **傳遞**：將價值交付給顧客。  
5. **售後**：提供後續服務。

### **收益流 (Revenue Streams) 與訂價機制**

* **產生方式**：資產銷售、使用費、會員費、租賃費、授權費、仲介費、廣告費。  
* **訂價策略**：  
  * **固定訂價**：統一價格、視產品特色、視客層、視數量。  
  * **動態訂價**：議價、收益管理（視庫存）、即時市場、拍賣。

### **關鍵資源與活動**

* **資源類別**：實體資源、智慧資源（品牌/專利）、人力資源、財務資源。  
* **活動類別**：生產（製造業）、解決問題（服務業）、平台/網絡（電商/社群）。

### **成本結構 (Cost Structure)**

* **成本驅動**：以降低成本為焦點（如低廉價格、自動化）。  
* **價值驅動**：以創造價值為焦點（如高價、高度個人化服務）。
